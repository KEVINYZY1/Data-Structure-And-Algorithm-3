

** Get Level of a node in a Binary Tree

Given a Binary Tree and a key, write a function that returns level of the key.

例如, 思考下图一颗二叉树, 如果输入一个数值k=3, 然后程序应该返回1. 如果输入的数值为4， 然后程序方法返回的数值为3. 如果输入的数值在二叉树中不存在，则应该返回0
